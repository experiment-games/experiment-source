# Bitwise Library

Contains functions for bitwise operations. Although Lua 5.4 has bitwise
operators these functions are still useful for compatibility with other
implementations of Lua.

## Functions

- [band](band.md)
- [bnot](bnot.md)
- [bor](bor.md)
- [bxor](bxor.md)
- [lshift](lshift.md)
- [rshift](rshift.md)
